# Data directory

The GitHub code repository does not include HDF5 data files.

Download the data and model archive from Zenodo or release assets, then place the data files here:

- `data/duo_synthetic_result.h5` for Fig. 5
- `data/duo_real_result.h5` for Fig. 7
- `data/Li_2022_16.h5` for the Li et al. comparison
- `data/gmt_data/*.gmt` for map boundary overlays
