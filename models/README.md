# Model directory

The GitHub code repository does not include trained model weight files.

Download the data and model archive from Zenodo or release assets, then place the model files here:

- `models/model_85000.pth`
- `models/model_sinv_65000.pth`

The included plotting scripts read processed HDF5 result files. The model weights are provided in the archive to record the trained model states used for the study.

https://doi.org/10.5281/zenodo.20174724
